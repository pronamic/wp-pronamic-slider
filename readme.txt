== Description ==

Pronamic block with slider integration.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/pronamic-slider` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress.
